## 📝 Known Limitations

⚠️ By Design (Demo Only):
- Passwords stored plaintext (not hashed)
- No real image upload (FileReader only)
- No payment processing
- No email sending
- Analytics data simulated
- No backend API
- Single browser/device (localStorage)

These can be enhanced with a backend for production use.
